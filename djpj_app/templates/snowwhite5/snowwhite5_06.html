조건문 if는 아래와 같은 형태를 지닙니다.
if [조건문] : [조건문이 진실일때 원하는 결과]
주의!: 만약 [조건문이 진실일때 원하는 결과]를 다음줄에 쓰고 싶을때는 반드시 들여쓰기를 해야합니다!
전체 문장에서 어떤 단어의 위치를 찾아보기 위해 find()를 이용할 수 있습니다.
그리고 문장에서 어떤 단어의 개수를 알아보기 위해 count()를 이용할 수 있습니다.

참고!: == 기호는 왼쪽과 오른쪽이 같은지 확인하라는 기호입니다. 주의!: 만약 [반복 하려는 내용]을 다음줄에 쓰고 싶을때는 반드시 들여쓰기를 해야합니다!

예시 조건문
리스트 중 한 단어에서 '독'이라는 단어가 있는지 찾고, 있으면 '독' 이라는 글자 출력하고, 없으면 '없다' 출력하기.
apples = ['독이있는사과', '깨끗한사과', '독을숨긴사과', '맛있는사과', '맛없는사과']
apple = '독이있는사과' number = apple.count('독') if number == 0:     print('독') else:     print('없다')
